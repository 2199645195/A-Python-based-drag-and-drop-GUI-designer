
# Mini Designer v20 使用手册
> **工业级轻量 GUI 设计器 | 拖拽设计 · 实时仿真 · 代码生成 · AI 协同**

## 📖 1. 概述
Mini Designer 是一款专为工业监控与仿真场景设计的纯 Python GUI 设计器。它解决了通用 Qt Designer 在工业开发中“图表难配、模板缺失、数据绑定繁琐”的痛点。

### 核心能力
-   **🏭 工业控件库**：内置多通道实时趋势曲线、图片背景框、PID 面板、报警灯组等专用组件。
-   **🔬 数据模拟器**：无需连接 PLC/后端，即可用正弦波、随机游走等模式驱动 UI 验证动态效果。
-   **💻 双模代码生成**：支持生成独立窗口类 (`GeneratedWindow`) 或 Ui 类 (`Ui_MainWindow`)，自带数据绑定管理器。
-   **🤖 MCP 自动化接口**：支持 AI 助手通过 JSON 指令直接操控画布（添加控件、截图、修改属性）。
-   **↩️ 完整编辑体验**：撤销/重做、智能吸附、多选批量操作、控件树拖拽改层级。

## 🎨 3. 核心功能详解

### 3.1 控件拖拽与布局
-   **添加控件**: 从左侧工具箱拖拽到画布，或双击工具箱项。
-   **智能吸附**: 移动控件时自动显示红色(边缘对齐)、绿色(等间距)、蓝色(居中)辅助线。
-   **多选操作**: `Ctrl+点击` 或框选多个控件，支持批量移动、对齐、删除、样式粘贴。
-   **层级调整**: 在「📁 控件树」面板中拖拽节点即可改变父子关系和 Z-order。
-   **微调位置**: 选中后用 `方向键` 像素级移动（开启网格时按格移动）。

### 3.2 📈 工业趋势曲线 (MultiTrendWidget)
专为焊接监控等场景设计的实时曲线控件：
-   **属性配置**: 双击曲线控件，在属性面板调整 `x_range`(时间窗口)、`y_min/y_max`、`auto_scale_y`、网格密度、配色。
-   **数据推送**: 通过 `addValue(channel_index, value)` 方法注入数据。
-   **演示模式**: 点击「🔍 预览」或数据模拟器，自动播放正弦波演示动画。
-   **代码生成**: 导出时自动嵌入完整类定义，无需额外依赖。

### 3.3 🖼️ 图片背景框 (ImageBgWidget)
用于设备图纸/P&ID 管路图叠加控件：
-   **设置背景**: 属性面板填写 `image_path`，或通过 MCP 指令设置。
-   **缩放模式**: 支持 `fit`(适配)、`stretch`(拉伸)、`tile`(平铺)、`center`(居中)。
-   **子控件叠加**: 本身是容器，可直接在其内部拖放按钮、标签等控件。
-   **Base64 内嵌**: 导出代码时自动将图片转为 Base64，实现单文件部署。

### 3.4 🔬 数据模拟器
**无需后端即可验证 UI 动态响应**：
1.  切换到右侧「🔬 模拟器」面板，点击「🔄 刷新」扫描所有带 Tag 的控件。
2.  为每个 Tag 选择波动模式：手动 / 正弦波 / 随机游走 / 锯齿波。
3.  设置参数：`min=最小值 max=最大值 T=周期(秒)`。
4.  点击「▶ 开始模拟」，观察曲线、进度条、数值标签的实时变化。

### 3.5 🏭 工业模板库
点击工具栏「🏭 模板」一键插入预设组件：
-   **暗色监控面板**: 温度/压力数值 + 进度条
-   **PID 控制面板**: SV/PV/MV 显示 + Kp/Ki/Kd 参数输入
-   **泵阀管路图**: 泵状态 + 阀门指示 + 流量压力
-   **报警指示灯组**: 正常/警告/故障三级指示
-   **设备状态卡片**: 运行状态 + 操作按钮
-   **用户自定义模板**: 选中控件 → 右键 → 「📦 另存为模板」

### 3.6 ✏️ 信号槽与回调编辑
1.  右键控件 → 「🔗 信号/槽编辑器」。
2.  选择源控件、信号、槽函数名，点击「➕ 添加连接」。
3.  右键连接项 → 「✏️ 编辑回调函数」，在弹出的代码编辑器中编写 Python 逻辑。
4.  生成的代码会自动缩进并替换 objectName 为实际变量名。

### 3.7 🎨 可视化 QSS 构建器
-   双击属性面板中的 `styleSheet` 字段。
-   在弹出窗口中通过颜色选择器、数值滑块调整样式。
-   左侧实时预览效果，下方显示生成的 QSS 代码。
-   点击「✅ 应用」写入控件，支持撤销。

## 💾 4. 项目管理

### 4.1 保存与加载
-   **保存**: `Ctrl+S` → 保存为 `.json` 项目文件（含所有控件、信号槽、回调代码、页面配置）。
-   **打开**: `Ctrl+O` → 加载 `.json` 项目文件，自动恢复全部状态。
-   **新建**: `Ctrl+N` → 清空画布（有未保存内容时提示确认）。

### 4.2 多页面管理
-   画布上方标签栏支持多页面设计。
-   「＋ 新建页」添加页面，双击标签重命名，点击 ⨉ 删除。
-   导出时自动生成 `QStackedWidget` + 侧边导航栏代码。

### 4.3 导出项目骨架
点击「📦 导出」→ 输入项目名 → 选择目录，自动生成：
```
my_project/
├── my_project.py        # 可运行的 Python 代码
├── my_project.json      # 设计器项目文件（可重新打开编辑）
├── my_project.qss       # 样式表
├── requirements.txt     # 依赖清单
├── opc_comm.py          # 数据绑定通信模板（如有 Tag 绑定时生成）
└── [自定义控件.py]      # 自动复制引用的自定义控件文件
```

## 🤖 5. MCP 自动化接口 (AI 协同)

设计器每 300ms 轮询 `gui_command.json`，支持外部脚本/AI 直接操控。

### 5.1 支持的指令
| action | params | 说明 |
| :--- | :--- | :--- |
| `add_widget` | `widget_type, x, y, w, h, text, role, tag` | 添加控件 |
| `delete_widget` | `name` | 按 objectName 删除 |
| `move_widget` | `name, x, y` | 移动控件 |
| `resize_widget` | `name, w, h` | 调整尺寸 |
| `set_property` | `name, property, value` | 修改任意属性 |
| `capture` | - | 截图保存到 `canvas_preview.png` |
| `get_canvas_info` | - | 获取画布尺寸、控件列表 |
| `set_canvas_size` | `w, h` | 设置画布尺寸 |
| `set_canvas_bg` | `path, scale, opacity` | 设置画布背景图 |
| `new_project` | - | 新建空白项目 |
| `save` | - | 保存当前项目 |

### 5.2 调用示例
```python
import json, time

cmd = {
    "id": "cmd_001",
    "action": "add_widget",
    "params": {
        "widget_type": "实时趋势曲线",
        "x": 50, "y": 50, "w": 400, "h": 250,
        "tag": "weld_temp"
    }
}

with open("gui_command.json", "w") as f:
    json.dump(cmd, f)

# 等待响应
time.sleep(0.5)
with open("gui_response.json", "r") as f:
    print(json.load(f))
```

## 🔧 7. 自定义控件扩展

### 7.1 加载外部控件
1.  编写 Python 文件，定义继承 `QWidget` 的类。
2.  可选：设置 `_display_name = "显示名称"` 属性。
3.  点击工具栏「🧩 自定义控件」→ 「📂 添加文件」。
4.  控件自动出现在工具箱「⭐ 自定义控件」分类中。

### 7.2 导入 SVG 图元
-   在自定义控件管理器中点击「🎨 导入 SVG」。
-   选择 `.svg` 文件，自动注册为可拖拽控件。
-   导出时自动内嵌 SVG 数据，无外部文件依赖。

## ❓ 8. 常见问题

**Q: 趋势曲线在高频数据下卡顿？**  
A: 当前使用 QPainter 绘制，建议数据点 >5000 时替换为 `pyqtgraph`。可在属性面板调整 `x_range` 减少可见点数。

**Q: 生成的代码如何接入真实数据？**  
A: 导出的 `opc_comm.py` 提供了 `DataBinder` 模板。在你的通信线程中调用 `self.data_binder.update_tag(tag, value)` 即可自动刷新对应控件。

**Q: 如何在论文中使用设计器截图？**  
A: 使用 MCP 的 `capture` 指令或直接 `F5` 预览后系统截图。预览模式隐藏所有编辑辅助元素，呈现纯净运行效果。

**Q: 自定义控件属性无法保存/恢复？**  
A: 确保属性可通过 `setProperty()` / `property()` 访问，且值为 JSON 可序列化类型。复杂对象需在 `_collect_widget_info` 中手动处理。
