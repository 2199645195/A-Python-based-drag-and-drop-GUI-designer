## 每个文件的详细职责

### `config.py` — 纯常量，零依赖
所有全局配置都在这一个文件里：
- **QSS 主题** — `DEFAULT_QSS`（亮色）、`DARK_QSS`（暗色），各 150+ 行样式规则
- **控件元信息** — `WIDGET_ROLES`（每个控件的可选角色）、`WIDGET_SIGNALS`（每个控件的信号列表）、`BINDABLE_WIDGETS`（支持数据绑定的控件类型）
- **控件目录** — `BUILTIN_CATEGORIES`（6 大分类 30+ 种内置控件）、`CUSTOM_WIDGETS`（动态加载的自定义控件）、`INDUSTRIAL_TEMPLATES`（6 套工业模板）
- **画布常量** — `MIME_TYPE`、`HANDLE_SIZE`、`SNAP_THRESHOLD`、`GRID_SIZE`、`DESIGN_WIDTH/HEIGHT` 等
- **工具函数** — `_esc()`（字符串转义）、`_sanitize()`（名称净化）、`_policy_to_str()`（QSizePolicy 枚举转字符串）

### `commands.py` — 撤销/重做命令模式
纯 Qt 类，不依赖其他模块：
- `Command` — 抽象基类（`execute()` / `undo()`）
- `AddWidgetCmd` / `DeleteWidgetCmd` — 添加/删除控件
- `MoveWidgetCmd` / `ResizeWidgetCmd` — 移动/缩放
- `PropertyChangeCmd` / `BatchPropertyChangeCmd` — 单属性/批量属性修改
- `BatchAlignCmd` — 多选对齐
- `ReorderWidgetCmd` — 容器内排序
- `ExtractWidgetCmd` — 从容器移出
- `InsertTemplateCmd` — 插入模板
- `_ReparentIntoContainerCmd` / `_ReparentBetweenContainersCmd` — 树拖拽跨容器移动
- `HistoryManager` — 撤销栈管理（push/undo/redo，上限 100 步）

### `codegen.py` — 代码生成器
依赖 `config.py`，将画布状态导出为可运行的 Python 代码：
- `CodeGenerator.generate()` — 主入口，生成完整 `main.py` 文件内容
- `CodeGenerator._generate_ui_class()` — 生成 `Ui_MainWindow` 类（pyuic 风格）
- 支持：多页面布局、数据绑定 `DataBinder`、信号/槽连接、锚点自适应 `resizeEvent`

### `canvas.py` — 画布核心（最大文件，~1800行）
依赖 `config`、`commands`、`codegen`、`dialogs/callback_editor`、`dialogs/signal_slot`：
- `DesignerCanvas` — 设计器画布 widget
- **控件生命周期**：创建（`_create_widget_internal`）、删除、复制粘贴、属性修改
- **交互**：鼠标拖拽移动/缩放、吸附对齐线、Ctrl+滚轮缩放、多选批量移动
- **渲染**：网格绘制、选中框、吸附线、插入指示器
- **多页面管理**：`add_page()` / `remove_page()` / `switch_page()` / `rename_page()`
- **项目序列化**：`save_project()` / `load_project()`（JSON 格式，含子控件递归）
- **预览模式**：`enter_preview_mode()` / `exit_preview_mode()`
- **右键菜单**：信号/槽、锁定/隐藏、复制粘贴样式、回调编辑
- `_Placeholder` — 画布空白时的占位 QLabel

### `app.py` — 主窗口
依赖所有模块：
- `DesignerMainWindow` — QMainWindow 子类
- **布局**：左右分栏（工具箱+控件树 | 画布+页面栏 | 属性+模拟器+历史）、底部代码预览区
- **工具栏**：对齐工具、撤销/重做、模板插入、暗色模式、主题编辑器、预览、保存/加载/导出
- **页面管理**：`QTabBar` 标签页（新建/关闭/重命名/拖拽排序）
- **持久化**：`QSettings` 保存窗口位置、分割比例、暗色模式、自定义控件列表

### 对话框 (`dialogs/`)

| 文件 | 类 | 功能 |
|---|---|---|
| `widget_loader.py` | `CustomWidgetLoader` | 动态扫描 `.py` 文件中的 QWidget 子类 |
| `theme_editor.py` | `ThemeEditorDialog` | QSS 编辑器，实时预览（300ms 防抖） |
| `callback_editor.py` | `CallbackEditorDialog` | 编辑信号槽的回调函数代码（Ctrl+S 保存） |
| `signal_slot.py` | `SignalSlotDialog` | 可视化连接信号/槽，支持右键菜单 |

### 面板 (`panels/`)

| 文件 | 类 | 功能 |
|---|---|---|
| `toolbox.py` | `WidgetToolbox` | 左侧控件列表，搜索过滤，拖拽到画布 |
| `widget_tree.py` | `WidgetTreePanel` + `_DragTree` | 控件层级树，拖拽改变父子关系 |
| `properties.py` | `PropertyEditor` | 属性表格，支持单控件/批量编辑 |
| `simulator.py` | `DataSimulatorPanel` | Tag 值模拟（正弦/随机/锯齿波） |
| `undo_history.py` | `UndoHistoryPanel` | 操作历史列表，点击可回退到任意步骤 |
